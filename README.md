# 氧化铝溶出工序数据发现与数据增强方法研究 (Alumina Dissolution Process Optimization)
本仓库包含了本科毕业设计《氧化铝溶出工序数据发现与数据增强方法研究》的完整核心算法实现代码。本项目针对拜耳法氧化铝溶出工序中存在的数据多源异步、缺失率高及高价值异常样本稀缺等问题，提供了一套基于机理驱动与数据驱动双轮并进的数据治理全生命周期工具链。
# 核心工作与创新点
1. **多速率数据对齐与异常清洗：** 采用 8 小时重采样视窗与防坍缩动态 Hampel 滤波，剔除高频异常飞点，实现多源异构数据的时空对齐。
2. **基于 HSIC 的特征优选：** 引入希尔伯特-施密特独立性检验 (HSIC)，量化复杂工艺变量间的非线性关联，剥离冗余信息，完成高维相空间的降维重构。
3. **融合物理机理约束的 TimeGAN 增强：** 在时序生成对抗网络 (TimeGAN) 中嵌入苛性比值限值、物料守恒等物理规则作为损失惩罚项，实现缺失数据的高保真插补与极端工况的合规生成。
4. **轻量化 1D-CNN 状态诊断：** 定制结合全局平均池化 (GAP) 的 1D-CNN 分类器，在极端工况下实现精准的工序状态预测。
